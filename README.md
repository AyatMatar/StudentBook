#  StudentBook - Interactive E-Book

StudentBook is an interactive web-based e-book designed to enhance student learning and engagement.  
It transforms traditional static content into an interactive experience using HTML5, CSS, and JavaScript.

---

##  Features

-  Display book content in a structured and user-friendly format
-  Audio playback for lessons (Play/Pause functionality)
-  Navigation between pages (Cover → Lesson → Quiz)
-  Interactive quiz with instant feedback
-  Simple and clean UI for better user experience

---

##  Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- jQuery (optional)

---

##  How It Works

1. The user starts from the **Cover Page**.
2. Clicks **Start** to navigate to the lesson.
3. The lesson includes:
   - Visual content
   - Audio explanation
4. The user moves to the **Quiz Page**.
5. Answers a multiple-choice question and gets instant feedback.
